## Installation

Install GO 1.21.5
```sh
https://go.dev/dl/
```

Install GO AIR 
```sh
go install github.com/cosmtrek/air@latest
```

Install all Package 
```sh
go get
```

Init AIR file 
```sh
air -c .air.toml
```

Run Project 
```sh
air
```
